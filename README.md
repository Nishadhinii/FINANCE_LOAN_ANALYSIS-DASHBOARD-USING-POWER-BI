# Finance Loan Analysis Dashboard

## Description
This project is an interactive Finance Loan Analysis Dashboard built using Power BI. It helps analyze loan data and provides meaningful insights about financial patterns such as income, loan amount, interest rate, and debt-to-income ratio (DTI).

The dashboard is designed to support better decision-making by visually representing complex data in a simple and understandable format.

## Objectives
- Analyze loan data based on different terms
- Compare financial metrics like income, loan amount, and interest rate
- Understand customer loan behavior
- Provide insights for financial analysis

## Dataset
The dataset used in this project includes:
- Annual Income
- Loan Amount
- Interest Rate
- Debt-to-Income Ratio (DTI)
- Loan Term (36 months / 60 months)
- Grade (A, B, C, D, E)
- Employee Length

## Dashboard Features

### Key Metrics
- Total Annual Income: 2.69 Billion
- Total Loan Amount: 436 Million
- Total DTI: 5.14K
- Total Interest Rate: 4.65K

### Visualizations
- Donut Chart – Installment Analysis
- Pie Chart – Annual Income by Term
- Bar Chart – Total Payment by Term
- Bar Chart – Interest Rate vs DTI
- Line Chart – Loan Amount by Term
- Donut Chart – Interest Rate Distribution

### Filters
- Loan Term
- Grade
- Employee Length

## Insights
- 36-month loans generate higher total payments
- Loan distribution varies across different grades
- Interest rate and DTI differ based on loan term
- Income concentration is higher in specific segments
